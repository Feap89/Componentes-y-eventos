# Pizzería Mamma Mía 🍕

Proyecto desarrollado como parte del **Desafío React – Hito 2**.  
El objetivo es construir una aplicación web con rutas, formularios controlados y validaciones básicas.

---

## 🚀 Tecnologías utilizadas
- [React](https://react.dev/) con [Vite](https://vitejs.dev/) como bundler
- React Router DOM para la navegación
- JavaScript (ES6+)
- CSS para estilos básicos

---

## 📝 Funcionalidades implementadas
- **Navegación** entre páginas: Inicio, Menú, Registro y Login.
- **Formularios controlados** con `useState`:
  - Registro: nombre, email, contraseña.
  - Login: email y contraseña.
- **Validaciones básicas**:
  - Campos obligatorios.
  - Contraseña con mínimo de 6 caracteres.
  - Mensajes de error y confirmación visibles.
- **Navbar** con enlaces a todas las páginas.
- **Footer** opcional para completar la estructura.
